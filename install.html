Installation manual

  • 1. Download .zip file with extension via this link.
  • 2. Open the file and copy magiword folder to some path on your PC.
  • 3. Open the browser and click options button in the top right corner.
  • 4. Go to More tools/Extensions.
  • 5. Turn on Developer mode in the top right corner.
  • 6. Press "Load unpacked" and choose path to magiword folder. Make shure to choose FOLDER, not some other file.
  • 7. Turn the extension on and enjoy learning new words with MagiWord!
